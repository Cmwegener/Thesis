Testing out a way to remove outliers from the bouy data. For this attempt I’m going to look at the salinity data from the EOS bouy. Here I am defining outliers as salinity values 3 standard deviations from the monthly salinity average. I tried this method looking at daily salinity values but some outliers remained and this way looks better. It still looks like some outliers might be present. I could reduce removing values 3 standard deviations away to 2 but I’m not sure what the best method of proceeding is.

EOS bouy data

eos<-read.csv("C:/Users/chels/Box Sync/Thesis/Data/Working data/Bouy data/eos_bouy_r.csv", header = TRUE, sep=",", fileEncoding="UTF-8-BOM", stringsAsFactors = FALSE)

eos$date<-as.Date(eos$date, format = "%m/%d/%Y")
eos$month_yr <- format(as.Date(eos$date), "%Y-%m")

Calculate monthly salinity average and sd

sal_monthly_av<-aggregate( salinity ~ month_yr,eos, mean)
sal_monthly_sd<-aggregate(salinity~month_yr,eos, sd)

Change column names

names(sal_monthly_av)[names(sal_monthly_av) == "salinity"] <- "monthly_sal_av"
names(sal_monthly_sd)[names(sal_monthly_sd) == "salinity"] <- "monthly_sal_sd"

Combine data frames

eos_sal_summary<-cbind(sal_monthly_av, sal_monthly_sd)

Add the monthly average and sd columns to the eos df

eos_new<-merge(eos, eos_sal_summary[, c("month_yr", "monthly_sal_av", "monthly_sal_sd")], by="month_yr")

Caluculate 3 standard deviations from monthly average and then remove salinity values that are greater.

eos_new$remove<-(3*eos_new$monthly_sal_sd)+eos_new$monthly_sal_av

removed<-filter(eos_new, salinity<=remove)

Graph comparing all data from new filtered data

removed<-ggplot(data = removed, mapping = aes(x = date, y = salinity, color=salinity)) + geom_point()  +stat_smooth(colour="green")+ggtitle("Outliers Removed") + theme(plot.title = element_text(size = 12, face = "plain")) +theme(axis.text=element_text(size=12),axis.title=element_text(size=12,face="plain")) + ylab("Salinity") + xlab("") + theme(panel.grid.major = element_blank(), panel.grid.minor = element_blank(), panel.background = element_blank(), axis.line = element_line(colour = "black"))
